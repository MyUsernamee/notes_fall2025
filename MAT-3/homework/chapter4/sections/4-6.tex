Let $S$ be the standard basis for $R^3$, and let $B = \{v_1, v_2, v_3\}$ be the basis in which $v_1 = (1, 2, 1)$, $v_2 = (2, 5, 0)$, and $v_3 = (3, 3, 8)$.
\subsubsection*{ 9 a) }

Find the transition matrix $P_{B \to S}$ by inspection.
\vfill


\subsubsection*{ b) }

Use Formula (14) to find the transition matrix $P_{S \to B}$.
\vfill

\pagebreak

\subsubsection*{ c) }

\vfill


\subsubsection*{ d) }

\subsubsection*{ Formulas }

(11)
\[
\begin{aligned}
[v]_B &= P_{B' \to B}[v]_B'
\end{aligned}
\]

(12)
\[
\begin{aligned}
[v]_B' &= P_{B \to B'}[v]_B'
\end{aligned}
\]

(14)
\[ [\text{new basis} \mid \text{old basis}] \quad \stackrel{\text{row operations}}{\longrightarrow} \quad [I \mid \text{transition from old to new}] \]
